### Pinterest

* Trouver le **Board ID** (méthode rapide). ([YouTube][1])
* **Pinterest API – Getting Access** (création d’app + authentification). ([YouTube][2])
* Comprendre l’**URL de redirection OAuth** Pinterest (utile pour les tokens). ([YouTube][3])

### TikTok

* Créer une **app TikTok Business** & obtenir les **access tokens**. ([YouTube][4])
* Implémenter le **login TikTok** et gérer les **access tokens**. ([YouTube][5])
* **TikTok Ads/API – Getting access** (vue d’ensemble de l’accès développeur). ([YouTube][6])

### Blog PrestaShop

* **Générer une clé API PrestaShop** (FR – pas à pas). ([YouTube][7])
* **Créer une clé API de webservice et y accéder** (FR). ([YouTube][8])
* **PrestaShop 8 – Webservices (API)** (cours/explications en FR). ([YouTube][9])
